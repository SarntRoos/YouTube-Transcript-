YouTube Transcript Fetcher

This Python script allows you to extract transcripts from YouTube videos, save them in a human-readable format, and store them in a specified folder as a JSON file. The script uses the youtube-transcript-api library to fetch the transcript and includes basic input validation for video URLs and folder names.

Features

Extracts transcripts from YouTube videos using their URL.
Saves transcripts in a JSON file in a specified folder.
Automatically formats the transcript for readability.
Validates YouTube video URLs and folder names to prevent invalid input.
Handles errors such as disabled transcripts or too many requests to the YouTube API.

Requirements

  -Python 3.x

  -youtube-transcript-api library
